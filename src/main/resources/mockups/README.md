# Compliance UI Mockups

Static HTML wireframes for the screens described in [product-spec.md](../../../../docs/product-spec.md).

## Viewing

With the dev server running:

```bash
npm start
```

Open **http://localhost:8080/mockups/** (index lists all screens).

Or open `index.html` directly in a browser from this folder.

## Screens

| URL (via `npm start`) | File | Requirements |
|----------------------|------|--------------|
| `/mockups/dashboard.html` | dashboard.html | F2.1 |
| `/mockups/sessions.html` | sessions.html | F2.2, F2.4 |
